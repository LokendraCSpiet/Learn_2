HTML Fundamentals

Overview

HTML drives the web. Whether you use Chrome, Firefox, Safari, or some other browser, many of us use the web daily. In this course you will learn the fundamental elements of creating web pages using HTML.

Not covered in this course is detailed information about CSS, JavaScript, or HTTP as those topics are covered in detail in other Abdullah courses1,2.

Objectives

  • Explain the fundamental concepts of HTML
  • Have fun
    • creating documents
    • Creating memories
  • mark it up!
